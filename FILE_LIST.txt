===========
1. Install .NET 8.0 SDK and MySQL
2. Create database: CREATE DATABASE Shop;
3. Update connection string in Services/DatabaseService.cs
4. Run: dotnet restore && dotnet run
5. Open: https://localhost:5001

For detailed instructions, see QUICK_START_GUIDE.md
